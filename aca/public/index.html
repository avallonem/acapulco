Automatic Construction of Clustered Parallel Coordinates Graph

Description

Acapulco a is tool for building clustered parallel coordinates graphs from security data obtained through the hpfeeds protocol. It was initially developed as part of the Google Summer of Code 2012 for The Honeynet Project. Acapulco uses Splunk as data managment backend and the powerful d3 visualization library.

Access the graph display tab, log into the Splunk server and hit the run button to build the graph from indexed data.
